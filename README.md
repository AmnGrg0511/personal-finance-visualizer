# Personal Finance Visualizer

Next.js app for tracking income, expenses, and budgets with clean charts and a structured transaction flow. Built to visualize trends and compare month-over-month performance.

## Highlights
- Transaction capture and categorization
- Budget planning with monthly rollups
- Recharts-based visuals for trends and breakdowns
- Light/dark theming with Radix UI primitives

## Tech Stack
- Next.js 15 (App Router)
- React 19
- Tailwind CSS
- MongoDB + Mongoose (data layer)
- Recharts
